========================================
 sfx/keys/  -  typing sounds
========================================

Drop ANY number of short sound files in this folder. Names don't matter here -
the mod uses every audio file it finds, and plays a RANDOM one on each keypress
(that's what gives the mechanical-keyboard variety). More files = more variety.

Accepted types: .mp3  .wav  .ogg  .m4a

WHAT'S HERE NOW
  key1.wav, key2.wav, key3.wav, key4.wav
      Default click sounds I generated so typing works out of the box.
      Delete them if you only want your own sounds.

TIPS
  - Keep them SHORT (a real key click is ~30-60ms) so fast typing doesn't overlap
    into mush.
  - Want one single click instead of random variety? Just leave one file in here.
  - After adding/removing files, click the toolbar speaker button and press "Reload"
    (or restart Zen).

NOTE ON SCOPE
  These fire for the browser's OWN text fields (address bar, search, find bar).
  They do NOT play while typing inside web pages - that needs a separate opt-in that
  listens for keystrokes on every site (sound-only, never records what you type).
  Ask if you want that added.
